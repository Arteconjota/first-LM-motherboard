Componentes de una Placa Base

nombre descripción rango de precio imagen
Zócalo Recibe también el nombre de socket y es ahí donde se encastra la CPU correspondiente. Es muy importante que ambos compartan la nomenclatura y sean compatibles. Es el conector donde se instala el procesador (CPU). Entre 80€ y 500€
Chipset Controla la comunicación entre la CPU, la RAM y otros componentes como los periféricos. Es el centro de comunicaciones de la placa base. Es el que controla y organiza los datos entre el procesador y el resto de componentes para que todo funcione correctamente. Entre 100€ y 500€
Disipadores Ayudan al resto de componentes que puede haber en la placa base a evacual el calor residual. Entre 20€ y 100€
Ranuras de Memoria RAM Permiten instalar módulos de memoria RAM en la placa base.Varía de 2 a 8 (las placas base para usuarios comunes suelen tener 4). Desde 32 GB hasta 128 GB o más en placas avanzadas. Entre 20€ y 1.000€
Ranuras de expansión PCIe Permiten instalar tarjetas de expansión como tarjetas gráficas, tarjetas de red o tarjetas de sonido. Entre 150€ y 400€
Conectores SATA y M.2 Se usan para conectar unidades de almacenamiento, como discos duros (HDD) o unidades de estado sólido (SSD). Las placas modernas suelen tener al menos 1 o 2 ranuras M.2. Entre 120€ y 400€
BIOS/UEFI Es el sistema que permite que la placa base se comunique con el hardware al encender el ordenador. Ofrece una interfaz gráfica más avanzada y mayor compatibilidad con discos duros grandes y sistemas de 64 bits. Integrado en la placa base, sin un precio específico por separado.
Puertos de Entrada/Salida Son los puertos que permiten conectar dispositivos externos como monitores, teclados, ratones, USB, etc. Entre 80€ y 400€ (Dependiendo de la cantidad y tipos de puertos)
Sistema de Alimentación (VRM) Regula y suministra energía de manera estable a la CPU y otros componentes. Las placas con un VRM de mejor calidad ofrecen mejor estabilidad, especialmente en sistemas de alto rendimiento. Entre 150€ y 500€
